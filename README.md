# Insertion Sort Assignment

## Overview

This project demonstrates the implementation and analysis of the Insertion Sort algorithm using Python.

## Features

- Insertion Sort implementation
- Best-case analysis
- Worst-case analysis
- Average-case analysis
- Stability demonstration
- Time complexity discussion
- Space complexity discussion

## Requirements

Python 3.x

## Running the Program

Open a terminal and run:

```bash
python insertion_sort.py
```

or

```bash
python3 insertion_sort.py
```
